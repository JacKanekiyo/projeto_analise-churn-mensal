# projeto_analise-churn-mensal
Análise preditiva e financeira de Churn (cancelamentos de clientes). O script processa históricos de ordens de serviço e dados contratuais para identificar falhas técnicas recorrentes, calcular o tempo de vida do cliente e projetar o impacto financeiro real no caixa. Desenvolvido em Python (Pandas e Seaborn).
